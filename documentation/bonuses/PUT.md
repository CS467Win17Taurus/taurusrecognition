# Bonuses: PUT

## /bonuses

Deactivate bonus amount in database

## URL Query Example
```
http://138.197.7.194/api/bonuses/
```

## Request Body
- **id** *(required)*: Database id of bonus

## Response

The response will include an object with status and message:

```
{"status":"success"}
```
```
{"status":"failed"}
```
