# Divisions: PUT

## /divisions

Deactivate division in database

## URL Query Example

```
http://138.197.7.194/api/divisions/
```

## Request Body
- **id** *(required)*: Database id of division

## Response

The response will include an object with status and message:

### JSON Objects
```
{"status":"success"}
```
```
{"status":"failed"}
```
